📱 AI-Based Remote Health Monitoring App

This Flutter application is part of an AI-powered wearable health monitoring system designed for real-time tracking and intelligent analysis of vital health parameters. The app is intended for both patients and caregivers, enabling continuous monitoring and rapid response in case of abnormalities.

⸻

🔍 Features

🩺 Real-Time Health Monitoring
	•	Live visualization of vital parameters:
	•	Heart Rate (BPM)
	•	SpO₂ (Blood Oxygen Saturation)
	•	ECG Signals
	•	Body Temperature

⸻

🤖 AI-Based Disease Prediction

The system predicts potential heart-related conditions using AI models:
	•	Bradycardia with Hypothermia
	•	Hypertensive Crisis with Tachycardia
	•	Normal Condition Detection

⸻

🚨 Motion & Safety Monitoring
	•	Motion anomaly detection
	•	Real-time fall detection and alerts

⸻

👥 User Support
	•	Designed for both end users (patients) and caregivers
	•	Enables remote monitoring and timely intervention

⸻

⚙️ Tech Stack
	•	Frontend: Flutter
	•	Backend: Firebase
	•	AI Model: XGBoost (Cloud Deployment)
	•	Hardware Integration: ESP32-based wearable sensors

⸻

📌 Summary

This application provides a smart, real-time, and AI-driven healthcare monitoring solution, improving early detection, safety, and remote patient care.
